# 해커톤 규칙

심화 트랙 참가자에게 공통으로 적용되는 규칙입니다. 자기 과제 폴더에 들어가기 전에 한 번만 훑어 주세요.

---

## ⏰ 1. 시간

- **타임박스: 2시간 · 혼자 완주**
- 시작 · 종료 시각은 강사가 공지합니다
- 각 과제설명서의 **"완성 기준"** 체크리스트를 기준으로 스스로 점검하세요
- 시간 이후 변경하신 기록은 Vercel 앱에 기록되므로 **시간 이후 수정은 추후 수상 확인 시 적발 및 수상 취소 되실 수 있습니다.**

---

## 🤖 2. 사용 가능한 AI 도구

- **Claude Code**: 본인의 **MAX 구독 계정 크레딧이 소진되기 전까지** 자유롭게 사용 가능합니다
- **앱 런타임 LLM 호출**: **Gemini API 만** 허용합니다 (OpenAI · Anthropic API · 기타 LLM 직접 호출 금지)
- **Gemini API 키**: 심화 트랙은 에이전트 수가 많고 호출량이 크기 때문에 각자 [Google AI Studio](https://aistudio.google.com/) 에서 **본인 계정으로 새 키를 발급**해서 사용하세요 (무료, 5분)
- **키 저장**: 반드시 `.env` 에. 코드 · Git 커밋 · 공개 채널에 절대 올리지 마세요

---

## 📊 3. 사용 가능한 데이터

- **제공 데이터만 사용**: 과제 폴더 안의 `./data/` 에 들어 있는 CSV · TXT · JSON 이 전부입니다
- 외부 보험 상품 API · 타사 데이터셋 · 웹 크롤링 금지

---

## 🧩 4. 결과물 원칙

- **환각 금지**: 화면에 뜨는 상품명 · 고객 ID · 약관 인용은 반드시 `./data/` 안에 **실제로 존재하는 값**만 써 주세요
- LLM 이 지어낸 가짜 데이터를 진짜 결과처럼 표시하면 완성으로 치지 않습니다

---

## 🛠 5. 오픈소스 라이브러리 · 프레임워크 (심화 트랙은 적극 활용 권장)

실력 발휘 포인트입니다. 단일 HTML 에 묶일 필요 없이, 원하는 스택으로 자유롭게 설계하세요.
